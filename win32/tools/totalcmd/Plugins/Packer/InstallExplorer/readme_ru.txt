InstallExplorer - архиваторный плагин для TotalCommander
========================================================================

Версия 0.9.1

Римейк плагина неизвестного автора "InstallExporer Port", использующий
исходный плагин InstallExplorer для FAR (© Sergei Wanin). Данный
архиваторный плагин позволяет просматривать и извлекать содержимое .exe и
.msi файлов, созданных инсталляторами Wise Installer, Vise Installer, Inno
Setup, Gentee Installer, InstallShield нескольких версий, NullSoft Installer
версий от 1.1o, SetupFactory, Eschalon Installer и Windows Installer (MSI).
Моя версия лишена недочетов старого плагина: частых падений при распаковке и
распаковки не тех файлов, которые были затребованы. В архиве находится
исходный плагин для FAR - InstallExplorer 0.3 от Sergei Wanin.

Установка - автоматическая (привязывается к расширению .msi).
Для открытия .exe файлов входите в них через <Ctrl+PageDown>
Внимание: с большими по объему файлами плагин работает медленно!
