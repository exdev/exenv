Плагин AudioInfo для Total Commander
------------------------------------
Данный content-плагин позволяет показывать в колонках панелей Total Commander
различную информацию об аудио-файлах.
Поддерживаемые потоковые форматы:
  MPEG (MP3/MP2/MP1), OGG, WMA, WAV, VQF, AAC, APE, MPC, FLAC, CDA.
Поддерживаемые трекерные форматы:
  IT, XM, S3M, MTM, MOD, UMX.

Поля, показываемые для потоковых форматов:
  Каналы, Время, Частота, Битрейт, Тип битрейта, Название, Исполнитель, Альбом,
  Трек, Дата, Жанр, Комментарий, Композитор, Авторское право, Ссылка, Кодировщик;
для трекерных форматов:
  Каналы, Время, Название.


Установка
---------
1. При наличии TC 6.50+ просто откройте архив, и TC сам установит плагин.
   Если до этого у Вас была установлена предыдущая версия плагина, то вначале
   удалите ее (для того чтобы при установке обновить "detect string" плагина).

2. Перейдите в меню Конфигурация -> Настройка -> Наборы колонок;
   добавьте новый набор, назовите его "Аудио-файлы" и добавьте в него несколько
   колонок; для каждой колонки нажмите "+" и выберите поле из списка AudioInfo.

3. Включите настроенный набор колонок:
   Вид -> Пользовательский набор колонок -> Аудио-файлы;
   наконец перейдите в каталог со звуковыми файлами.

-----------------
06.12.05: добавлена поддержка CDA, поле "Комментарий" показывает позицию трека;
          добавлено поле "Теги", показывающее, какие теги присутствуют в файле
05.12.05: поправлено отображение битрейта для WAV, AAC
22.11.05: поправлена поддержка MP3, APE;
          поле "Частота" переименовано в "Битрейт";
          добавлено поле "Тип битрейта" (поддерживается для MP3, WMA, AAC)
12.11.05: добавлена поддержка трекерных форматов: IT, XM, S3M, MTM, MOD, UMX;
          добавлена поддержка FLAC и многоканальных WAV;
          добавлены поля: "Трек (ведущий ноль)", "Авторское право", "Ссылка",
          "Весь текст" (позволяет производить полнотекстовый поиск);
          поле "Поставщик" переименовано в "Кодировщик"
09.05.05: добавлена поддержка MPC
05.05.05: поправлена поддержка WMA
09.02.04: добавлена поддержка APE
30.01.05: поправлено отображение времени при длительности >1 часа;
          добавлено поле "Время (Ч/М/С)"
03.11.04: добавлена поддержка MPEG, WAV, WMA, AAC, VQF
27.10.04: время показывается в Ч:М:С, битрейт в Kbps, частота в Гц/КГц
26.10.04: первоначальная версия с поддержкой OGG
